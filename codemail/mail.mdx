## Get Started

- Select a username or text to report as if to copy it.

- Open the extension by clicking the icon in your extensions toolbar.

- Then hit "Generate Report".

## Data it Uses

- Current text selected
- What page you're on
- If the page has Zymono attributes
- Any images on the page

## Common Errors

<AccordionGroup>
  <Accordion title="This website's owner has requested to block Zymono.">
    The owner of the website blocked the Zymono Chrome extension.
  </Accordion>
  
  <Accordion title="Cannot audit a report on chrome:// pages.">
    Chrome prevents extensions from running code on any official Google/Chrome pages.
  </Accordion>
