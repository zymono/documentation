  <Accordion title="Add a Zymono button">
    Use The Zymono API To Embed A Button Into Any Webpage That Launches Your Zymono Report Page.

  ```html reportButton.html
    <script type="module" id="zymono-script">
    //THIS CODE WAS MADE FOR THE ZYMONO API AND FOLLOWS OUR API LICENSE. 
    //https://github.com/zymono/API/LICENSE.md
    
    //https://github.com/zymono/API
    
    window.sessionStorage.setItem('gameID', 'Your_Panel_ID_HERE')
    
    import { createButton, report } from 'https://api.zymono.com/api.js'

    createButton()
</script>
 ```
  </Accordion>

  <Accordion title="How to find panel ID">
    Go to https://zymono.com/me/ and under Panel Key, click `Click to reveal key`. Copy the following key to your clipboard!
  </Accordion>

  <Accordion title="API safety">
    Make sure to store all instances of your panel key in an environment variable. Your panel key must remain hidden otherwise users can access the API on your behalf.
  </Accordion>
   <Accordion title="What's the API">
    The Zymono API gives you the ability to turn any iframe into a Zymono panel where you can manage reports without having every user authenticate and go through the collaboration process.
  </Accordion>
