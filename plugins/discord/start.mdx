# Setup

- Install the Discord bot [here](https://discord.com/api/oauth2/authorize?client_id=1088884665026347089&permissions=8&scope=bot).
- Go to https://zymono.com/me/ and copy your Panel Key
- Once you are in your Discord server, run %setKey {Your Panel Key}

# How To File A Report

Run: `/report {user} {reason}`.
