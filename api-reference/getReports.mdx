---
title: "Get Reports"
description: "Retrieve your Zymono reports."
api: "POST https://zymono.com/api/getReports"
---

### Body

<ParamField body="uid" type="string">
  Your Panel Key. (Keep this secret!)
</ParamField>

### Response

<ResponseField name="Reports" type="array">
  An array with JSON objects inside it which include the ID and other information.
</ResponseField>

<RequestExample>

```js Example Request
const response = await fetch('/api/getReports', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ 'Your_Panel_Key' }),
  });

  if (response.ok) {
    const reports = await response.json();
    console.log(reports)
    reports.forEach(report => {
      console.log(report)
    })
  } else {
    console.error('Error fetching reports:', response.status);
  }
```

</RequestExample>
