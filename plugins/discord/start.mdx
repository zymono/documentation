# Setup

- Download the Discord bot at: https://discord.com/api/oauth2/authorize?client_id=1088884665026347089&permissions=8&scope=bot
- Copy your Panel Key from https://zymono.com/me/
- Run %setKey `Your Panel Key` in Discord chat

Done!

# How To File A Report

Run: `/report {user} {reason}`
