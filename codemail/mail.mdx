## Create Mailing List

<CodeGroup>

```html cm.html
<head>
  <script src="https://cdn.emailjs.com/sdk/2.3.2/email.min.js"></script>
</head>
  
<script type="module">
    import { initializeApp } from "https://www.gstatic.com/firebasejs/9.14.0/firebase-app.js";
    import { getStorage, ref, uploadString } from "https://email-1.zymono.repl.co/1.0/storageapi.js"; 
</script>

<form id="cmForm">
  <input placeholder="Please Enter Your Email" id="codemail-email" type="email" required><br>
  <input type="submit" value="Subscribe"><br><br>
  This email signup form was created with Zymono Codemail and the Zymono <a href="//zymono.com/policy/privacy.html">Privacy Policy</a> and <a href="//zymono.com/policy/terms.html">Terms of Service</a> apply.
</form>

<script type="module">
  import { subscribe, saveCloud } from  "https://codemail.zymono.com/1.0/mail.js"
  window.addEventListener("load", function() {
    const cmt = 'Your_Mail_Token_Here'
    document.getElementById('cmForm').addEventListener("submit", function(event) {
      event.preventDefault()
      subscribe(cmt, 'FORM_NAME_HERE')
    })
  })
</script>
```
</CodeGroup>

Navigate back to https://codemail.zymono.com/dashboard/ to see and export your mailing list.

## Zymono Fire

<Card title="Watch Video" icon="youtube" iconType="duotone" href="https://www.youtube.com/watch?v=o89aeL--xLY">
  Watch tutorial on YouTube.
</Card>

<CodeGroup>
```html authHandler.html
<script>
  window.localStorage.setItem('displayName', new URLSearchParams(location.search).get('n'))
  window.localStorage.setItem('email', new URLSearchParams(location.search).get('email'))
  window.localStorage.setItem('uid', new URLSearchParams(location.search).get('uid'))
  window.localStorage.setItem('photoURL', new URLSearchParams(location.search).get('pfp'))
</script>
```
```html redirect.html
  <script>
    window.location.href = "//fire.zymono.com?b={mySite}/authhandler.html"
  </script>
```
</CodeGroup>

## Common Errors

<AccordionGroup>
  <Accordion title="Error when redirecting back from Zymono Fire.">
    Fix by renaming your file from `authHandler.html` to `index.html` and placing it in a folder called `authHandler` and updating `redirect.html`.
  </Accordion>
